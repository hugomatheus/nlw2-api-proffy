# Proffy
> Proffy API | Next Level Week 2 - Rocketseat.


## Installation

```bash
# Clone this repository
$ git clone https://github.com/hugomatheus/nlw2-api-proffy.git

# Go into the repository
$ cd nlw2-api-proffy

# Install dependencies
$ yarn install

# Run migrates
$ yarn knex:migrate

# Start server
$ yarn start
```

## Technologies

* Nodejs
* Express
* Knex
